# 🛒 NXT-Mart - E-Commerce Platform

## 📌 Overview

NXT-Mart is a full-stack e-commerce web application built to provide a seamless online shopping experience. Users can browse products, view details, manage their cart, and perform authentication-based actions through a responsive and modern interface.

## 🚀 Features

* User Authentication & Authorization
* Product Listing and Details
* Shopping Cart Functionality
* Responsive UI Design
* Secure Backend APIs
* MongoDB Database Integration
* JWT-Based Authentication
* React Router Navigation

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* React Router DOM
* Tailwind CSS
* Hero Icons

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* CORS
* Dotenv

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd nxt-mart-master
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=8000
DB_NAME=nxtmart
MONGO_URL=your_mongodb_connection_string
```
